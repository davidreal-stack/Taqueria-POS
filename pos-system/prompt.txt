Debe ser un MVP funcional pero con arquitectura profesional y modular para escalar en el futuro.

# 🧱 TECNOLOGÍA OBLIGATORIA

* Backend: Python con Flask
* Base de datos: SQLite (archivo local)
* ORM: SQLAlchemy
* Frontend: HTML + CSS + JavaScript (sin frameworks pesados)
* El sistema debe correr localmente en:
  http://localhost:5000

# 💻 EJECUCIÓN

* Debe incluir instrucciones claras para ejecutar en VS Code
* Debe poder abrirse desde un solo comando:

2. Cajero

   * crear órdenes
   * cobrar

3. Cocina
